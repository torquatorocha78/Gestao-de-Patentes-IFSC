# Gestao-de-Patentes-IFSC
📂 Gestão de Patentes – IFSC  Sistema de gestão de patentes desenvolvido em Python para organizar o cadastro, consulta e acompanhamento de pedidos de patente no Instituto Federal de Santa Catarina (IFSC).  Essa aplicação foi feita para facilitar o controle interno de patentes, gestão dos pagamentos, o app criado em python para deploy em streamlit
